FibX is the ultimate DeFi execution layer and pre-built skill framework designed for AI agents. It enables AI agents to autonomously perform complex multi-chain financial operations through natural language. Supported operations include token swaps (powered by Fibrous aggregation for optimal rates), Aave V3 position management (supply, borrow, repay), asset transfers, and transaction verification across Base, Citrea, HyperEVM, and Monad. FibX operates via a machine-readable CLI that outputs structured JSON. Developers can interact directly via terminal (e.g., npx fibx trade 0.1 ETH USDC --chain base) or plug our ready-to-use Agent Skills library into any LLM framework using npx skills add Fibrous-Finance/fibx-skills. It features built-in authentication (Private Key or Privy email OTP) and simulates every transaction before broadcasting to prevent failed executions. Connecting AI to DeFi usually requires custom integrations, managing complex wallet infrastructure, and writing brittle on-chain scripts by hand. FibX handles all the heavy lifting RPC endpoints, gas mechanisms, token approvals, and safety guardrails natively. FibX CLI and skills framework are 100% open-source and free to integrate. Standard network gas fees apply for on-chain executions.
